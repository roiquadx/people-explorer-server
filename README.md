````md
# People Explorer - Server (Node.js + TypeScript)

Minimal Express API with TypeScript and lightweight persistence.

## Prerequisites

-   Node.js 18+
-   pnpm

## Install & Run

```bash
pnpm install
pnpm dev
```
````

Server runs on: http://localhost:3001

## Build & Run (Production-like)

```bash
pnpm install
pnpm build
pnpm start
```
